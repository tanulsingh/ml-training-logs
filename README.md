# ML Training Logs

Code companion to [tanulsingh.github.io](https://tanulsingh.github.io) — from-scratch implementations and experiments for blog posts and paper explanations.

## Structure

```
blog/           # Maps 1:1 to blog posts
  01-tokenization/  → /blog/tokenization
papers/         # Maps to paper explanations
  deberta/          → /papers/deberta
common/         # Shared utilities
```

## Conventions

- PyTorch for all implementations
- Code is self-contained and runnable
- Notebooks follow: explanation → implementation → results
